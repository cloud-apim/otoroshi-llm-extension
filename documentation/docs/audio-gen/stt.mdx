# Speech-to-text (STT)

## Supported providers

* OpenAI
  * whisper-1
  * gpt-4o-mini-transcribe
* Groq
  * whisper-large-v3
* Mistral
  * voxtral-mini-latest
  * voxtral-mini-2507

## Configuration

```js
{
  "connection": {
    "token": "${vault://local/OPENAI_API_TOKEN}",
    "timeout": 30000
  },
  "options": {
    "tts": {
      "enabled": true,
      "model": "gpt-4o-mini-tts",
      "response_format": "mp3",
      "speed": 1
    },
    "stt": {
      "enabled": true,
      "model": "gpt-4o-mini-transcribe"
    },
    "translation": {
      "enabled": true,
      "model": "whisper-1"
    }
  }
}
```

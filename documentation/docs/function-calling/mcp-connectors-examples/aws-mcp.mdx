# AWS KB Retrieval

### Set up secret API Key

Before setting up your MCP connector entity, you can follow [this guide](/docs/llm-gateway/secrets) to add your secrets `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY` and `AWS_REGION` to your local Otoroshi vault.

### Configuration

```js
{
 "command": "npx",
  "args": [
    "-y",
    "@modelcontextprotocol/server-aws-kb-retrieval"
  ],
  "env": {
    "AWS_ACCESS_KEY_ID": "${vault://local/AWS_ACCESS_KEY_ID}",
    "AWS_SECRET_ACCESS_KEY": "${vault://local/AWS_SECRET_ACCESS_KEY}",
    "AWS_REGION": "${vault://local/AWS_REGION}"
  }
}
```

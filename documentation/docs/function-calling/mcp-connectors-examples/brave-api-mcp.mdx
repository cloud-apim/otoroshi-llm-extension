# Brave Search API

### Set up secret API Key

Before setting up your MCP connector entity, you can follow [this guide](/docs/llm-gateway/secrets) to add a secret `BRAVE_API_KEY` to your local Otoroshi vault.

### Configuration

```js
{
  "command": "npx",
  "args": [
    "-y",
    "@modelcontextprotocol/server-brave-search"
  ],
  "env": {
    "BRAVE_API_KEY": "${vault://local/BRAVE_API_KEY}"
  }
}
```

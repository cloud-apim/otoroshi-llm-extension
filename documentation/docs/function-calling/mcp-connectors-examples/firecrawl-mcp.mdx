# Firecrawl

### Set up secret API Key

Before setting up your MCP connector entity, you can follow [this guide](/docs/llm-gateway/secrets) to add a secret `FIRECRAWL_API_KEY` to your local Otoroshi vault.

### Configuration

```js
{
  "command": "npx",
  "args": [
    "-y",
    "firecrawl-mcp"
  ],
  "env": {
    "FIRECRAWL_API_KEY": "${vault://local/FIRECRAWL_API_KEY}"
  }
}
```

# Install

## Download Otoroshi

[ ![Download](https://img.shields.io/github/release/MAIF/otoroshi.svg) ](https://github.com/MAIF/otoroshi/releases/download/v16.24.0/otoroshi.jar)

First, download the Otoroshi jar file:

```sh
curl -L -o otoroshi.jar 'https://github.com/MAIF/otoroshi/releases/download/v16.24.0/otoroshi.jar'
```

## Download the LLM Extension

[ ![Download the latest release of the Otoroshi LLM Extension](https://img.shields.io/github/release/cloud-apim/otoroshi-llm-extension.svg) ](https://github.com/cloud-apim/otoroshi-llm-extension/releases/download/0.0.42/otoroshi-llm-extension_2.12-0.0.42.jar)

Download the latest release of the `Otoroshi LLM Extension` from [here](https://github.com/cloud-apim/otoroshi-llm-extension/releases/latest).

```sh
curl -L -o otoroshi-llm-extension.jar 'https://github.com/cloud-apim/otoroshi-llm-extension/releases/download/0.0.42/otoroshi-llm-extension_2.12-0.0.42.jar'
```

## Run Otoroshi with the LLM Extension

**WARNING**: the Otoroshi LLM Extension only run on **JDK 17** and above

Run Otoroshi with the LLM extension by executing the following command :

```sh
java -cp "./otoroshi-llm-extension.jar:./otoroshi.jar" -Dotoroshi.adminLogin=admin -Dotoroshi.adminPassword=password -Dotoroshi.storage=file play.core.server.ProdServerStart
```

This will start the Otoroshi API Gateway.

You can access the Otoroshi UI by opening [http://otoroshi.oto.tools:8080/](http://otoroshi.oto.tools:8080/) in your browser.

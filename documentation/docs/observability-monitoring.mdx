# Observability & Monitoring

Observability & Monitoring provide insights into LLM interactions, performance, and security. Every request is logged, analyzed, and can be exported for further reporting.

Key Features

Real-time Monitoring: Track LLM usage, response times, and errors.

Detailed Auditing: Log request metadata, consumer details, and LLM provider usage.

Exportable Logs: Forward logs to third-party services like Elasticsearch, Datadog, Prometheus, etc.

Anomaly Detection: Identify unusual request patterns, potential security threats, or performance degradation.

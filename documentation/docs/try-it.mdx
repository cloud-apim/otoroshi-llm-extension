# Try Otoroshi LLM Extension

there is currently several ways to try the Otoroshi LLM Extension

## On premise

just follow the [install page](./install.mdx)

## On Cloud APIM Otoroshi instances

order a new Otoroshi instance from the [Cloud APIM Console](https://www.cloud-apim.com/otoroshi-managed). The LLM Extension is included in every plan.

## On Clever Cloud

order a new Otoroshi addon from the [Clever Cloud Console](https://www.clever-cloud.com/developers/doc/addons/otoroshi/). The LLM Extension is included in every plan.


## On Cloud APIM Serverless

If you want to try Otoroshi LLM Extension for free, you can test it for free on [Cloud APIM Serverless](https://www.cloud-apim.com/serverless).

You can use this [project template](https://github.com/cloud-apim/otoroshi-llm-extension-serverless-example). All the explainations are available on [our blog](https://blog.cloud-apim.com/unleash-the-power-of-your-api-gateway-with-ai-a-simple-guide-to-create-your-first-ai-gateway)
